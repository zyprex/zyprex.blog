Connect Internet From Command Line¶

The <SSID> is your wifi name.

On Windows¶

Connect to WIFI: netsh wlan connect name=<SSID>

Disconnect to WIFI: netsh wlan disconnect

On Linux¶

Check WIFI status: nmcli radio wifi, it should output enabled, or else you will need run nmcli radio wifi on.

Check Network status: nmcli d status, nmcli c show

List WIFI SSID: mncli d wifi list , mncli d wifi rescan

Delete a WIFI: nmcli c delete <SSID>, nmcli c delete <UUID>

Connect to new WIFI: nmcli d wifi connect "<SSID>" --ask

Connect to hidden WIFI: nmcli d wifi connect "<SSID>" hidden yes --ask

Connect to WIFI: nmcli c up <SSID>, or nmcli c up

Disconnect to WIFI: nmcli c down <SSID>, or nmcli c down
