  • color (optional)

    • true

Timer¶

parameter¶

  • id
  • time (optional) UTC time string, for example: “2020-01-01 12:12:12”
  • format (optional) %y=years, %d=days, %h=hours, %m=minutes, %s=seconds, %e=milliseconds
  • interval (optional) default 1ms

usage¶

{{ < timer
  id="mytimer_test"
  format="%y Year %d Day %h:%m:%s.%e"
  interval="80" >}}
