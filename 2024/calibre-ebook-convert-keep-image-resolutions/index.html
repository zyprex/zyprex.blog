Calibre Ebook Convert Keep Image Resolutions¶

When I convert ebook use calibre’s ebook-convert command, I discover the output file’s image size has been downscaled.

This can be avoid by add an argument.

ebook-convert mybook.mobi mybook.epub --output-profile tablet
