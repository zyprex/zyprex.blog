Useful Chrome Flags¶

To Access the flags, type chrome://flags in browser address bar (omnibox), then search those word and enable them.

  • forced color
  • reader mode
  • smooth scrolling
  • GPU rasterization
  • zero-copy rasterization
  • turn off caching of streaming media to disk
  • back-forward caching
  • QUIC protocol
  • parallel downloading
  • download range support
