How the Vim Save My Life
2023/08/29 Tue 12:30

Prologue¶

I use vim editor about 2 years before. There are so many guides in the web. I think it’s not my duties to proclaimed the powerful of vim editor. But the vim editor really save my life from a mistake of file deletion.

The Bitter After Disaster¶

When I delete files in an accident, I feel the whole world swirled around me, and angry, agony filled my heart. It’s suffered a lot. I try the Windows File Recovery, but it’s recover other files and just not the files I deleted a moment ago.

If you curious my steps, I use winfr E: C: /regular /n *.*, it’s works somehow, but for me is not enough.

Soon I realized the vim’s backup file, and successful get lost files. Admirable vim!, all is thanks to I edit files in vim.

The Vim Editor¶

Do you know how to set the backup for vim ?

Type :e $MYVIMRC, and write those line.

set backup backupdir=~/.cache/vim/backup,$TMP/vim/backup.

It’s useful when you delete files that you just edited by accident. You can find the files in ~/.cache/vim/backup/, with a ~ suffix.

Epilogue¶
