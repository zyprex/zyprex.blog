term explainÂ¶

  • page: a page can be a markdown file or one section,taxonomy,term …
    • kind
    • home: index.html
    • page: page not in a directory, viz. md file in content root directory
    • section: page in a directory
    • taxonomy: default taxonomy is categories and tags
    • term: each category or tags name
  • type: determine render template and used archetype template
