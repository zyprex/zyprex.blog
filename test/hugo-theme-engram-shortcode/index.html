hugo shortcode intro¬∂

some useful shortcode I made for hugo theme engram.

What’s a shortcode¬∂

the shortcode is a piece of html defined in THEME/layouts/shortcode/

A simple example¬∂

1
2
3

<div class="{{.Get "class"}}">
  {{- .Inner -}}
</div>

this file located in shortcode folder and it’s named d.html

then at your page, you can write below line. NOTE: don’t keep any space between ‘{’ and ‘<’

1
2
3

{{ <d class="a-c">}}
some center text
{{ </d>}}

below is effect:

some center text

for official guide, please see hugo shortcode-templates or hugo build-in shortcode

highlight shortcode (hugo build-in)¬∂

code:

{{ < highlight go "linenos=table,hl_lines=1-2 4,linenostart=199" >}}
package main
import "fmt"
func main() {
    fmt.Println("hello highlight")
}
{{ < / highlight >}}

phrase¬∂

code:

{{ <phrase>}}
quick fox jump over the lazy dog  
quick fox jump over the lazy dog  
quick fox jump **over** the lazy dog
{{ </phrase>}}

result:

quick fox jump over the lazy dog
quick fox jump over the lazy dog
quick fox jump over the lazy dog

avatar¬∂

NOTE: this will use config.toml params avatar

code:

{{ <avatar slogan="just do it">}}

result:

just do it

tooltip¬∂

code:

{{ <tooltip "this is tooltip content, it can very long or use raw html">}} here {{ </tooltip>}}

result:

mouse over here ( on mobile device click it)

tabs¬∂

code:

{{ <tabs "one"
    "tab1" "tab2" "tab3"
    "tab1 **content** ---"
    "tab2 content +++"
    "tab3 content ***"
> }}
{{ <tabs "two"
   "tab4" "tab5" "tab6"
   "tab4 content ~~~"
   "tab5 content ==="
   "tab6 content ###"
> }}

result:

tab1 content —
tab2 content +++
tab3 content **
tab4 content ~~~
tab5 content ===
tab6 content ###

fold¬∂

code:

{{ <fold open="false" title="this ...">}}
this fold is hidden by default, if open params is "false" or it absent.
{{ </fold>}}

{{ <fold open="true">}}
this fold is open by default, cause the open params is "true"
{{ </fold>}}

result:

 this ...
 this ...
this fold is hidden by default, if open params is “false” or it absent.
 
 
this fold is open by default, cause the open params is “true”
