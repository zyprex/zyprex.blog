Remap Capslock on Windows¶

Dual key remap is a more clean, safe and powerful way to remap the almost useless capslock key.

Use ‘dual key remap’¶

The project home: dual-key-remap ↗

Auto Start ‘dual key remap’ when windows launch¶

Make a ‘dual-key-remap.exe’ shortcut (by press Alt and drag to somewhere) and put it in Startup folder (win + R and input shell:startup you can directly open this folder).

Stop ‘dual key remap’¶

C:\>tasklist | findstr dual
dual-key-remap.exe            1556 Console                    4      4,148 K
C:\>taskkill /pid 1556

Or: taskkill /im dual-key-remap.exe
