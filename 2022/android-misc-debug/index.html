App Crashed When use android:theme="@android:style/Theme.Dialog"¶

For Android Studio set android theme, in AndroidManifest.xml, correct the android:theme value:

1
2
3
4

<activity
    android:name=".DialogActivity"
    android:theme="@style/Theme.AppCompat.Dialog"
    android:exported="false"/>

‘com.android.support.percent’ with AndroidX¶

The new androidx standard didn’t compatible with old support packcage.

无法导入‘com.android.support.percent‘(百分比布局依赖库)的解决方法 ↗

‘getAdapterPosition’ always return -1¶

Move it into onBindViewHolder function.

dependencies {
  implementation fileTree(dir: 'libs', includes: ['*.jar'])
}

On building APK¶

Generate Signed APK: Errors while building APK. You can find the errors in the ‘Messages’ view.

Enlarge jvm max heap to 256m (I origin set it to 128m)

org.gradle.jvmargs=-Xmx256m -Dfile.encoding=UTF-8
