Git¶

install git ↗

know terms¶

NameDesc.
working directory (tree)your files entity
stashstore latest changes which may leads to conflict
staging area (stage or index)diff record
repository (repo)contains all working fruits for you
SHA (commit id)unique hash commit id

help your self¶

git <cmd> --help view command’s detail.

set up personal information¶

git config --global user.name "your name"
git config --global user.email "your email"
git config --global core.editor "vim"
git config --global color.ui true

save your password¶

git config --global credential.helper store
git config --global credential.helper 'cache --timeout=3600'

first commit¶

git init
# create [file], edit it
git add [file]
git commit -m "first commit"
git commit -v

file ops¶

git add .
git rm [file] # remove from tree
git rm --cached [file] # remove from stage
git rm -f [file] # remove from stage and tree
git mv [old] [new]
git commit -m "message"

see what is changed¶

git status
git diff [file] # diff from stage
git diff [--cached|--staged][HEAD|SHA]
git diff [SHA1|SHA2]
git log --pretty=oneline
git blame <file> [-L line1,+len]

undo and redo¶

git checkout -- [file] # check out from stage
git commit -ammend -m 'Fixxes bug #43'
git reflog # show SHA
git reset --soft  [SHA] # repo -> SHA
git reset --mixed [SHA] # stage & repo -> SHA
git reset --hard  [SHA] # tree & stage & repo -> SHA

branch¶

new branch¶

git branch [new] [SHA|tag]
git checkout -b [new] [SHA|tag]

switch to branch¶

git branch [-a]
git checkout [branch]
git switch [branch]

sort branch¶

git branch -m [new_name]
git merge [other_newer]
git rebase [branch]
git rebase [branch] [to_branch]
git cherry-pick [SHA1] [SHA2] [SHA3]
git rebase -i [SHA]

delete branch¶

git branch -d [which]
git push -d origin [remote_branch]

temp branch¶

git stash
git switch [branch_B]
# ... do sth.
git switch [branch_A]
git stash pop
git stash list
git stash clear
git stash pop stash@{2}

Github¶

register in github ↗

link to remote repo¶

  1. create new repo on github.
  2. add one commit in your local repo.
git remote add origin main
git push -u origin main

Then, next time use git push origin main

get new update from remote repo¶

git fetch       # Update 'origin master' from remote
git pull        # = git fetch + git merge
git pull --all  # Fetch all remotes

regret after push¶

git revert [SHA]
git push [-all]

tag¶

git log --pretty=oneline --abbrev-commit
git tag                            # local tags
git ls-remote --tag                # remote tags
git tag <tagname> [-m"<msg>"][SHA] # create tag
git tag -d <del_tagname>           # del tag
git push origin <tagname>          # push tag
git push origin --tags             # push all tag
git push -d origin <remote_tag>    # del tags

Problem¶

git add¶

Error:

LF will be replaced by CRLF Causes: use unix format file on windows. Solution: use tool “unix2dos”, convert LF to CRLF. or change autocrlf settings:

git config --global core.autocrlf false # default is true

git rm¶

Error:

git rm - fatal: pathspec did not match any files
Causes: unknow.
Solution:

git add .
git rm -f [files]

git remote add¶

Error:

git remote add origin <your_git:account> fatal: remote origin already exists. Causes: unknow Solution:

git remote rm origin

git pull¶

Error:

commit your changes or stash them before you can merge. Causes: local file contains unpushed changes. Solution:

  1. stash them
git stash
git pull
git stash pop
git diff
  1. discard local changes
git reset --hard
git pull

git push¶

Error:

git push origin master error: failed to push some refs to Causes: forgot to synchronized remote repo before push. Solution:

  1. use git push -f.
  2. merge from remote repo
git pull --rebase origin main
git push -u main

Error:

fatal: could not read Username for ‘https://github.com’: No such file or directory Causes: credential expired! Solution: reinstall credential or upgrade git or use ssh.

Error:

fatal: unable to access ‘https://github.com/NNN/XXXX.git/': Failed to connect to 172.0.0.7 port YYYY: Timed out Causes: may thanks to GFW :) Solution: time is the key. git clone then git push.

Error:

Failed to connect to github.com port 443: Timed out Causes: also thanks to GFW :) Solution: do it later.

git rebase -i¶

Error:

main|rebase (1|1) Causes: your rebase action is on the way. Solution:

git add .
git rebase --continue

or use rebase --abort exit rebase
